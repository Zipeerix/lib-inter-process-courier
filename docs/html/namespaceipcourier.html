◆ makePayloadFromProto()

template<IsDerivedFromProtoMessage ProtoType>
SerializedProtoPayload ipcourier::makePayloadFromProto ( const ProtoType & message)

Serializes a Protocol Buffer message into a payload string.

Takes a Protocol Buffer message, serializes it to binary format, and creates a payload string that includes both the message type information and the serialized data. The resulting payload can be transmitted over inter-process communication channels and later deserialized back to the original message type.

Template Parameters
ProtoTypeThe Protocol Buffer message type (must derive from google::protobuf::Message)
Parameters
messageThe Protocol Buffer message to serialize
Returns
std::string The formatted payload string containing type info and serialized data

◆ makeProtoFromPayload()

template<IsDerivedFromProtoMessage ProtoType>
ProtobufToolResult< ProtoType > ipcourier::makeProtoFromPayload ( const SerializedProtoPayload & payload)

Deserializes a payload string back into a Protocol Buffer message.

  • Parses a payload string created by makePayloadFromProto() and reconstructs the original Protocol Buffer message. The function validates that the type information in the payload matches the expected message type before attempting deserialization.
Template Parameters
ProtoTypeThe expected Protocol Buffer message type (must derive from Message)
Parameters
payloadThe payload string containing type info and serialized data
Returns
ProtobufToolResult<ProtoType> Either the deserialized message or an error code indicating the failure reason
Return values
ProtoTypeSuccessfully deserialized Protocol Buffer message
InvalidFormatThe payload string doesn't contain the expected delimiter
TypeMismatchThe type name in the payload doesn't match ProtoType
DeserializationFailedThe Protocol Buffer parsing failed
See also
makePayloadFromProto for the reverse operation
Example:
// Assuming MyMessage is a defined Protocol Buffer message
const SerializedProtoPayload payload = "my.package.MyMessage:some_binary_data";
const auto result = ipcourier::makeProtoFromPayload<MyMessage>(payload);
if (result) {
MyMessage msg = *result;
// Use msg
} else {
// Handle result.error()
}
